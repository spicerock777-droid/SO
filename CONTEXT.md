# CONTEXT

## カルテ

現場レポートの性格づけ。単発の作業報告ではなく、症状（現況・所見）→ 原因（複合要因の診断）→ 処置（応急対応）→ 経過・方針（今後の予定）を経時的に記録する診断書としての機能を持つ。エリアごと・訪問ごとに蓄積されることで、過去の状態と処置の履歴を参照できる記録になる。

## ご確認・ご判断いただきたい点

レポート内に設けるセクション。まずは自分自身の記録として、所見から見える根本対策（追加の土壌改善作業など）の要検討事項を書き留めておく。クライアントに見せる場面では、このセクションがそのまま実施可否・予算承認を仰ぐ問いかけとして機能する。状況報告と切り分けて扱う。
